Accessibility statement for Apply for a National Insurance number

This accessibility statement applies to the Apply for a National Insurance number website. It explains how accessible this website is, and what we are doing to improve accessibility.

This website is run by the Department for Work and Pensions. We want as many people as possible to be able to use this service. For example, that means you should be able to:

  • change colours, contrast levels and fonts
  • zoom in up to 300% without the text spilling off the screen
  • navigate most of the website using just a keyboard
  • navigate most of the website using speech recognition software
  • listen to most of the website using a screen reader (including the most recent versions of JAWS, NVDA and VoiceOver)

We’ve also made the text on this website as clear as possible.

AbilityNet has advice on making your device easier to use if you have a disability.



How accessible this website is:

Most parts of this website are fully accessible and can be accessed by either a screen reader or by a keyboard.

We know that some Dragon users may struggle to expand the links containing help text.

We also know that when applicants upload a photo, the Choose file button doesn’t respond to the Dragon voice command “Click button and say what you see.” Dragon users can tab to the Choose file button and activate it by using either of the voice commands Press Space bar or Press Enter.

Note that:

  • when you change the size of the browser window, the text will reflow in a single column
  • you can modify the line height or spacing of text
  • you can navigate all of the questions using a keyboard
  • you can skip to the main content when using a screen reader

Report accessibility problems with this website

We are always looking to improve the accessibility of this website. If you find any problems not listed on this page or if you think this website does not meet accessibility requirements, contact us

We aim to reply to you within 7 days.



Enforcement procedure

In England, Scotland and Wales, the Equality and Human Rights Commission (EHRC) is responsible for enforcing the Public Sector Bodies (Websites and Mobile Applications) (No. 2) Accessibility Regulations 2018 (the ‘accessibility regulations’). If you are not satisfied with our response to your complaint, contact the Equality Advisory and Support Service (EASS).

In Northern Ireland, the Equality Commission for Northern Ireland (ECNI) is responsible for enforcing the Public Sector Bodies (Websites and Mobile Applications) (No. 2) Accessibility Regulations 2018 (the ‘accessibility regulations’). If you’re not happy with how we respond to your complaint, contact the Equality Commission for Northern Ireland (ECNI).



Technical information

The Department for Work and Pensions (DWP) and the Department for Communities (DfC) are committed to making this website accessible, in accordance with the Public Sector Bodies (Websites and Mobile Applications) (No. 2) Accessibility Regulations 2018.



Compliance status

This website is partially compliant with the Web Content Accessibility Guidelines version 2.1 AA standard, due to the non-compliances listed below.

Non-accessible content

The content listed below is non-accessible for the following reasons.

Non-compliance with the accessibility regulations

When applicants upload a photo, the Choose file button doesn’t respond to the Dragon voice command “Click button and say what you see.” This fails WCAG 4.1.2 Name, Role, Value (A), 1.3.1 Info and Relationships (A) and WCAG 2.4.7 Focus Visible (AA).



This accessibility statement

This statement was prepared on 3 November 2020. It was last reviewed on 13 January 2022.

This website was last tested on 17 November 2021. The test was carried out by the Department for Work and Pensions using a combination of manual and automated testing, including assistive technology.

All pages within the Apply for a National Insurance Number journey were tested.
